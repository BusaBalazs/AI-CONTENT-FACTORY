# Platform Components

## 1. Scheduler

Responsible for triggering automated workflows.

Examples:

* Cron
* Webhooks
* Manual execution

---

## 2. Content Sources

Collects information from external systems.

Examples:

* RSS
* Reddit
* GitHub
* Hacker News
* Files
* APIs

---

## 3. Content Normalizer

Converts different input formats into a common internal structure.

---

## 4. AI Gateway

Provides a unified interface for all AI providers.

Priority:

1. Google Gemini
2. OpenRouter
3. Ollama

---

## 5. Content Processing

Transforms content using AI.

Examples:

* Summarization
* Translation
* Rewriting
* Classification
* Metadata extraction

---

## 6. Human Approval

Allows manual review before publishing.

---

## 7. Publishing

Publishes content to external platforms.

Examples:

* LinkedIn
* Facebook
* X
* WordPress
* YouTube

---

## 8. Analytics

Collects metrics and execution statistics.
